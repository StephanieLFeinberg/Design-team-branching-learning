# Branching practice repo

## Today we're going to learn Branching and merging.

1. Create a branch. Name it your name.
    Command **git branch branch-name**
2. Add a file. Name it your favorite seasion and initials like so **fall-sf**
3. Add whatever you choose to the file! your favorite seasonal activities, why you love that season, or just a string of random text.
4. Practice switching branches. 
    Command **git checkout branch-name**
5. Practice seeing all branches on your local. 
    Command **git branch**
6. Got to Github. Review and merge the pull request you created.
7. Delete your branch.
    Command **git branch -d branch name**

## Tips if you get stuck
- **ask cursor agent!**
   - "What is the command to create a branch?"
   - "What does a commit do?"
